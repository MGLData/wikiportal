DM Portal Questions

DM Portal is designed to support hundreds of users at once. In order to acheive this, many of the files are copied to your local harddrive when you first open the application. If Portal is closed incorrectly (froze, shutting down from the task manager, etc.) your machine may think Portal is still open because a file is locked "open". Using the 'Reset Port' file will typically resolve this issue; however, shutting down the machine completely will resolve the issue if the first avenue is unsuccessful.
